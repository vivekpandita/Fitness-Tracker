Setting up API:

* Set up Eclipse IDE
* Import Maven project where you select the project till \API\fitness-tracker
* Let it import completely.
* Open file -> com.bits.fitnesstracker.fitnesstracker.FitnessTrackerApplication
* Run as Java Application
* Spring boot server should start


For accessing DB:
- After Running Server, go to URL -> http://localhost:8080/h2-console
- Login with default creds . Just verify JDBC URL should be -> jdbc:h2:mem:testdb
- Click on Connect
